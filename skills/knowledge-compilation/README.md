# knowledge-compilation

Compiled-knowledge workflows for Obsidian-style vaults with clear source-space vs compiled-layer boundaries.

## Included skills
- `karpathy-llm-wiki-obsidian`
- `obsidian-inbox-to-wiki-ingest`
- `obsidian-wiki-lint-triage`

## Use this bundle when
- raw notes are accumulating without durable synthesis
- you need repeatable Inbox -> wiki compilation
- you want lint/triage for knowledge health rather than cosmetic formatting

## Core ideas
- source space and compiled knowledge are different layers
- every ingest should remain traceable to source material
- lint should improve navigability and knowledge quality, not just neatness
