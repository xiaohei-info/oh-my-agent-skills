# multi-agent-control

Controller-side patterns for subagent delegation, bounded handoff design, and verification.

## Included skills
- `subagent-first`
- `subagent-collaboration-workflow`

## Use this bundle when
- you are orchestrating multiple child agents
- delegation is causing drift or scope creep
- you need clearer controller vs implementer boundaries

## Core ideas
- delegation needs explicit contracts
- controllers keep judgment and verification
- parallelism only helps when dependencies are mapped first
