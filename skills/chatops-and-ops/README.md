# chatops-and-ops

Human-readable recurring automation patterns for chat delivery and operational monitoring.

## Included skills
- `user-friendly-cron-messages`
- `ops-sentry`

## Use this bundle when
- alerts are too noisy or repetitive
- cron outputs read like internal dumps instead of useful messages
- you need deduplicated operational monitoring

## Core ideas
- optimize for human scanning speed
- compress routine output aggressively
- escalate only when the user's interpretation or action changes
