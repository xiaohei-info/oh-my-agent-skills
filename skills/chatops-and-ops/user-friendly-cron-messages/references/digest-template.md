# Digest Template

Use this template for daily/weekly summaries, recaps, market briefs, research digests, or periodic review cron jobs.

## Routine Digest

```text
🟢 常规摘要｜<topic>
<一句话总结：今天/本轮整体无重大偏离，重点看以下几点>

📌 重点结论:
- <takeaway 1>
- <takeaway 2>
- <takeaway 3>

📊 关键事实:
- <fact 1>
- <fact 2>
- <fact 3>

🎯 当前判断:
- <one-line judgment / action stance>

⚠️ 主要风险:
- <single biggest risk>

👀 下一步关注:
- <watchpoint 1>
- <watchpoint 2>
```

## Expanded Digest

```text
🟡 需要关注｜<topic>
<1-2 句话说明：本轮与上轮相比，哪些点发生了实质变化>

📌 重点结论:
- <takeaway 1>
- <takeaway 2>
- <takeaway 3>
- <optional takeaway 4>

📊 关键事实:
- <fact 1>
- <fact 2>
- <fact 3>
- <fact 4>

🔥 变化点:
- <what changed>
- <why it matters>
- <what became more or less important>

🎯 当前判断:
- <updated judgment>

✅ 建议动作:
- <action 1>
- <action 2>

⚠️ 风险与不确定性:
- <risk 1>
- <optional risk 2>

👀 下一步关注:
- <watchpoint 1>
- <watchpoint 2>
- <watchpoint 3>
```

## Notes
- A digest is not a raw log dump.
- Keep only the most important 3-5 takeaways.
- Emphasize what changed since the last cycle.
- Avoid restating background unless it changed the conclusion.
