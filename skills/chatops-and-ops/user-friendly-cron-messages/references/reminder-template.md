# Reminder Template

Use this template for follow-up reminders, nudges, pending confirmations, scheduled check-ins, or user-action reminders.

## Simple Reminder

```text
🟢 提醒｜<topic>
<一句话说明现在为什么提醒你>

📌 当前事项:
- <what is due / pending>

✅ 现在最短动作:
- <single next step>

⚠️ 若暂不处理:
- <what happens / what remains blocked>
```

## Attention Reminder

```text
🟡 需要处理｜<topic>
<1-2 句话说明为什么这件事现在值得处理>

📌 当前事项:
- <pending item 1>
- <optional pending item 2>

🔥 为什么现在值得看:
- <reason 1>
- <reason 2>

✅ 建议动作:
- <action 1>
- <action 2>

📝 若需回复，可直接回:
- <minimal reply format>
```

## Urgent Reminder

```text
🔴 重要提醒｜<topic>
⚠️ <一句话说明紧急性来源>

📌 当前事项:
- <what needs attention now>

🔥 影响:
- <impact 1>
- <impact 2>

✅ 建议立即动作:
- <action 1>
- <action 2>

📝 最小回复格式:
- <reply format>
```

## Notes
- Reminders should minimize thinking cost.
- Focus on the next action, not long background.
- Ask for the smallest useful reply when user input is needed.
- If nothing is urgent, keep the tone light and short.
