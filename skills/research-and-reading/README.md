# research-and-reading

Tool-routing and deep-analysis methods for reading the web and producing structured research.

## Included skills
- `web-reading-router`
- `hv-analysis`

## Use this bundle when
- you need a reliable URL-reading escalation path
- research needs more than a shallow summary
- you want a reusable analysis framework instead of ad hoc note-taking

## Core ideas
- choose the narrowest reliable reading path first
- separate reading, extraction, and escalation clearly
- combine longitudinal and cross-sectional analysis for deeper judgment
