# Skills

This directory groups the public skill collection into a small number of thematic bundles.

## Bundles

- `engineering-execution/` — verification and debugging discipline
- `multi-agent-control/` — controller-side delegation workflows
- `skill-engineering/` — authoring, packaging, and optimization of skills
- `chatops-and-ops/` — chat-native cron UX and operational alerting
- `research-and-reading/` — URL routing and deep research workflows
- `knowledge-compilation/` — compiled-knowledge workflows for Obsidian-style vaults

## Packaging rule

One skill = one folder.

Within each skill folder:
- `SKILL.md` is the entrypoint
- support files stay beside the skill
- copy the whole folder when installing into another environment

## Start here

If you do not know where to begin:
- start with `engineering-execution/`
- then `multi-agent-control/`
- then `skill-engineering/`
