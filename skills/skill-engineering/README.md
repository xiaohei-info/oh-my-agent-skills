# skill-engineering

Methods for designing, packaging, improving, and publishing reusable skills.

## Included skills
- `skill-optimizer`
- `writing-skills`
- `external-hermes-skills-lifecycle`

## Use this bundle when
- you are authoring a new skill
- you want to package a local skill library publicly
- you want to improve skill quality systematically

## Core ideas
- optimize for discoverability first
- package support files correctly
- treat skill quality as something you can evaluate and iterate
