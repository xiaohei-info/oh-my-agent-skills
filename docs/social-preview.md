# Social Preview Assets

This repository includes generated social-preview images at:

- `assets/social-preview.png` — primary landscape preview for GitHub and general launch posts
- `assets/social-preview-xhs.jpg` — smaller Xiaohongshu-friendly version for creator-center publishing flows

## Intended use

Use these assets for:
- GitHub social preview settings
- launch posts
- repository announcements
- documentation previews

## Notes

- The assets are stored in-repo so future maintainers can reuse or replace them.
- If you want GitHub to use the landscape image as the repository social preview card, upload it in the repository settings UI.
- The images were generated specifically for `oh-my-agent-skills` and are meant to communicate:
  - installable agent skills
  - reusable workflow bundles
  - modular adoption across one-skill, one-bundle, or fuller installs
