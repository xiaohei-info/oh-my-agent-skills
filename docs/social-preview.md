# Social Preview Asset

This repository includes a generated social-preview image at:

- `assets/social-preview.png`

## Intended use

Use it for:
- GitHub social preview settings
- launch posts
- repository announcements
- documentation previews

## Notes

- The asset is stored in-repo so future maintainers can reuse or replace it.
- If you want GitHub to use this image as the repository social preview card, upload it in the repository settings UI.
- The image was generated specifically for `oh-my-agent-skills` and is meant to communicate:
  - agent methodologies
  - Hermes-compatible skills
  - reusable workflow bundles
